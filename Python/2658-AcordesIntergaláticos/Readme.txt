Antes dos acordes serem tocados, todas as teclas do piano estão associadas à mesma nota, que é a nota 1. 
Portanto, a lista das notas antes dos acordes serem tocados será uma lista com N elementos, todos iguais a 1.

Pode=se inicializar essa lista da seguinte maneira em Python:


notas_antes = [1] * N


Isso criará uma lista notas_antes com N elementos, todos iguais a 1.

Essa lista representa o estado inicial das notas associadas a cada tecla do piano antes de qualquer acorde ser tocado. 
Em seguida, à medida que cada acorde for tocado, o programa irá atualizar as notas associadas às teclas de acordo com as regras descritas na questão.
